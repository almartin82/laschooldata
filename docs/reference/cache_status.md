# View cache status

Shows which years have cached data.

## Usage

``` r
cache_status()
```

## Value

Data frame with cache status for each year

## Examples

``` r
cache_status()
#>   end_year tidy_cached wide_cached
#> 1     2019        TRUE        TRUE
#> 2     2020        TRUE        TRUE
#> 3     2021        TRUE        TRUE
#> 4     2022        TRUE        TRUE
#> 5     2023        TRUE        TRUE
#> 6     2024        TRUE        TRUE
```
