# View cache status

Shows which years have cached data.

## Usage

``` r
cache_status()
```

## Value

Data frame with cache status for each year

## Examples

``` r
cache_status()
#>   end_year tidy_cached wide_cached
#> 1     2019       FALSE       FALSE
#> 2     2020       FALSE       FALSE
#> 3     2021       FALSE       FALSE
#> 4     2022       FALSE       FALSE
#> 5     2023       FALSE       FALSE
#> 6     2024       FALSE       FALSE
```
